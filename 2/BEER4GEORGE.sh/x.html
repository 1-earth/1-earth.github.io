to contribute, please follow these rules:

shelldown@myr.sh.

shelldown

a partial but useful implementation of Markdown
in POSIX shell script that only depends on printf

release notes

v0.2.2: added support for linked images.
v0.2.1: fixed single word links and images.


usage

shelldown reads from standard input, so use shell
redirection or pipe content, e.g.

$ shelldown < file.md


installation

copy the shelldown script to your $PATH or source
it in your shell rc file.
